# Ebook Extractor

This skill turns ebook files into plain text an agent can work with.

Use it when you have an EPUB, MOBI, or PDF and need readable text for analysis, quoting, summarizing, indexing, or search. It is especially useful after a book has been found or downloaded, because the real work often starts after the file exists.

PDFs, EPUBs, and MOBIs all hide text in different ways. This skill gives the agent a simple path from "I have a book file" to "I can inspect the content."

Good fits:

- Extracting a chapter for study
- Preparing a book for search
- Pulling text into a research workflow
- Converting an ebook before asking questions about it

It does not use an LLM for extraction. It uses local tools, which keeps the first pass mechanical and repeatable. Extraction quality varies by file type and scan quality; OCR should be explicit, not assumed.
