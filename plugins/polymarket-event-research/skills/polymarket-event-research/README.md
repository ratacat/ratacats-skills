# Polymarket Event Research

This skill is for price-blind research on how a Polymarket-style event would actually resolve.

Use it when the user wants the real-world path: offices, courts, agencies, deadlines, procedures, proof standards, local context, and the facts that would make a market resolve Yes or No. It avoids odds, prices, volume, holders, and trading talk.

Many markets look simple until the resolution rule touches a messy institution. This skill helps the agent trace the path from headline to resolver without getting distracted by market movement.

Good fits:

- Understanding resolution mechanics
- Finding source authority
- Explaining legal or procedural paths
- Mapping actors and institutions
- Preparing a handoff for later market analysis

Use this before market-aware work when the basic world question is still cloudy. It is narrower than `pm-deep-analysis`: it focuses on the resolution path and source authority for one Polymarket-style event.
